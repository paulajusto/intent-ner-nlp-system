# intent-ner-nlp-system
Projecte de PLN amb classificació d’intencions i NER. S’hi implementen pipelines complets de preprocessament, embeddings, CNN, BiGRU i Transformers, amb anàlisi de hiperparàmetres, regularització i balanceig de classes. Avaluació rigorosa amb F1 macro/ponderada i comparació de models.
